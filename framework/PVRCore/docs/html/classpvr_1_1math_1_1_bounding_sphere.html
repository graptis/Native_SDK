void pvr::math::BoundingSphere::expand ( const glm::vec3 &  point)
inline

Expand the bounding sphere to includes the given point. If the point is not already inside the sphere, increase the radius and move the center so that the new sphere contains both the old one and the new point (the new sphere exactly contains the new point as is tangent to the old sphere point at the point opposite to the new point in relation to the center)

Parameters
pointThe new point to include
