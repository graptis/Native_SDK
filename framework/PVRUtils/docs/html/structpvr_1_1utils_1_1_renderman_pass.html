Get the commands necessary to render this entire Pass (for each subpass, for each node, bind pipeline, descriptor sets, draw commands etc.) into a Primary command buffer (secondary command buffers cannot be used at this level) Allows to configure if the begin/end renderpass and/or updateUniform commands will be recorded

Parameters
cbuffA Primary command buffer to record the commands into.
swapIdxThe current swap chain (framebuffer image) index to record commands for.
beginEndRendermanPassIf set to true, record a beginRenderPass() at the beginning and endRenderPass() at the end of this function. If loadOp is "clear". If the loadop is "clear", the first Model's clear colour will be used.

Get the commands necessary to render this entire Pass (for each subpass, for each node, bind pipeline, descriptor sets, draw commands etc.) into a Primary command buffer (secondary command buffers cannot be used at this level) Will call begin/end renderpass commands at start and finish, and allows to explicitly specify the clear color.

Parameters
cbuffA Primary command buffer to record the commands into.
swapIdxThe current swap chain (framebuffer image) index to record commands for.
clearColorThe color to which to clear the framebuffer. Ignored if loadop is not clear.

void pvr::utils::RendermanPass::updateAutomaticSemantics ( uint32_t  swapidx)
inline

Iterates all the nodes semantics per-effect, per-pass, per-subpass, per-model, per-node, and updates their values to the ones retrieved by the Model (model,mesh,node,material) retrieved values. createAutomaticSemantics needs to have been called before.

Parameters
swapidxThe current swapchain index (used to select which buffer to update in case of multibuffering. Multi-buffers and non multi-buffers are handled automatically, so always pass the current value).
