Multi<pvrvk::DescriptorSet> pvr::effect::PipelineDef::fixedDescSet[4]

A "Fixed" descriptor set is a descriptor set that does not export any semantics, and is set by the PFX. For example, a descriptor set containing only a texture "marble.pvr" which was set through the effect and has no semantic defined, would be fixed. A fixed descriptor set will exist in this field. Non-fixed set IDs will be null here.
