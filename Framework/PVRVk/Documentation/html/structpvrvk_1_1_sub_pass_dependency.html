Parameters
srcSubPassThe subpass for the source of the dependency
dstSubPassThe subpass for the destination of the dependency.
srcStageMaskA bitmask describing which stages of source must be completed before destination
dstStageMaskA bitmask describing which stages of the destination subpass must not start before source
srcAccessMaskA bitmask describing which kinds of operations of source must be completed before destination
dstAccessMaskA bitmask describing which kinds of operations of destination must be completed before source
dependencyByRegionSet to "true" if the dependencies are only relevant on a region by region base, set to false if it is possible that source calculations from a part of the image may be used in another part of the render in destination.
