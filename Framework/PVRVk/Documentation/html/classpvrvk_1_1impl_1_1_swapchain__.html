bool pvrvk::impl::Swapchain_::acquireNextImage ( uint64_t  timeOut,
Semaphore  signalSemaphore = Semaphore(),
Fence  signalFence = Fence() 
)

Acquire next image. The acquired swapchain index can be retrieved by calling getSwapchainIndex. Note: The presenation engine may still be consuming the swapchain image, therefore the calle must synchronise it before using it.

Parameters
timeOutindicates how long the function waits, in nanoseconds, if no image is available.
signalSemaphoreis null semaphore or a semaphore to signal
signalFenceis a null fence or fence to signal
Returns
Return true if no error occured
