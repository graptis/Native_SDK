Create a blending state. Separate color/alpha factors.

Parameters
blendEnableControls whether blending is enabled for the corresponding color attachment. If blending is not enabled, the source fragment’s color for that attachment is passed through unmodified. Default: false
srcBlendColorselects which blend factor is used to determine the source factors (Sr,Sg,Sb). Default: VkBlendFactor::e_ONE
dstBlendColorSelects which blend factor is used to determine the destination factors (Dr,Dg,Db). Default: VkBlendFactor::e_ZERO
srcBlendAlphaSelects which blend factor is used to determine the source factor Sa: VkBlendFactor::e_ONE
dstBlendAlphaSelects which blend factor is used to determine the destination factor Da. VkBlendFactor::e_ZERO
blendOpColorSelects which blend operation is used to calculate the RGB values to write to the color attachment. Default: VVkBlendOp::e_ADD
blendOpAlphaSelects which blend operation is use to calculate the alpha values to write to the color attachment. Default: VkBlendOp::e_ADD
channelWriteMaskBitmask of VkColorComponentFlagBits specifying which of the R, G, B, and/or A components are enabled for writing. Default: All
